# NoteApp 

NoteApp is a simple and clean **SwiftUI-based iOS application** that allows users to create, view, edit, and delete notes easily.

---

##  About The Project

NoteApp is built using **SwiftUI** focusing on clean UI and basic note-taking functionality.  
This app is useful for learning SwiftUI fundamentals like state management, navigation, and CRUD operations.

---

##  Features

- Create new notes
- View list of notes
- Edit existing notes
- Delete notes
- Simple & clean SwiftUI UI
- Real-time UI updates using SwiftUI state
